В каталоге "Stepic-analyse" собраны практические задачи и небольшие проекты, реализованные во время обучения на курсе
"Аналитик данных" на stepik, среда - pyCham, датасеты скачиваются из интернета, вывод в консоль.

Содержание:
1_taxi.py - задачки по поиску распределения заказов такси по райнам
2_taxi_peru.py - задачки по визуализации распределения оценок водителей и пешек
3_hotel.py - проект по анализу бронирования номеров в отелях (подробное описание в 3_hotel_readme.txt)
4_operations.py - проект по анализу двух датасетов о действиях пользователей на каком-то продукте (подробное описание в 4_operations_readme.txt)
